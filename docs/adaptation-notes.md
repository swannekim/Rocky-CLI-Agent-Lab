# Adaptation Notes

This repo is intentionally easy to fork into other workshop themes.

## What to swap out first

If you want a different persona, edit:

- `.github/agents/rocky-guide.agent.md`
- `.github/skills/rocky-voice/SKILL.md`

Keep:
- repository-wide structure
- mission brief format
- beginner explanation format
- facilitator flow

## Good alternative personas

- calm debugging mentor
- first-year CS tutor
- code review critic
- incident-response operator
- sci-fi mission controller
- friendly technical PM

## What should remain stable

- clarity
- compact structure
- one main voice idea
- good examples
- reusable prompts

## What usually breaks quality

- too many gimmicks
- too many catchphrases
- no output structure
- trying to imitate a character too literally

## Recommendation

Homage works better than impersonation.  
Strong structure works better than pure style.
